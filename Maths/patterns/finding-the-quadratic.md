# To find the quadratic where it goes through a specific point.
When a quadratic passes through the point $(x,y)$, use the fromat vertex form:
$$a(x-h)^2 +k =y$$
where $h$ = x-cordinate of the vertex of parabola and $k$ = y-cordinate of the vertex of parabola.

NOTE: do not confuse the coordinates of the point in which the polynomial passes through with the point of vertex.
Point passing through $= (x,y)$
point of vertex $= (h,k)$
## Touches the x axis or tangent to x axis:
since the parabola touches x-axis, then $k = 0$.
Using this in the format
$$a(x-h)^2 = y$$ 
gives us the required polynomial
## The vertex lies on the line $y = mx + c$
When there is a relationship between the parabola and a line, it can be solves with substituting the the value of k as $k = mx + c$
which gives us the format:
$$a(x-h)^2 +  (mx + c) = y$$ 

# Shifting the qudratic to some $x$ and $y$
Here vertex from is possible however it can get complicated thus we would substitute the following.
Since every point of the parabola shifts we can simply substitute the shift for the x and y value.
For a function $$f(x) = ax^2 + bx +c $$
let us transform everypoint by 2 units in x direction and -3 units in y direction. 
This takes place in 
$$f(x)= a(x-2)^2 + b(x-2) + c -3$$
as c represents the y value of parabola and a and b show the x value.
